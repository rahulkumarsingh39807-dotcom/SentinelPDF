# 🛡️ SentinelPDF

**SentinelPDF** is a professional static PDF malware analysis platform built using **Python** and **Flask**.

The application helps security analysts inspect suspicious PDF files without executing them by performing static analysis such as metadata extraction, structure analysis, JavaScript detection, IOC extraction, and risk scoring.

---

## ✨ Features

* 🔐 User Authentication
* 📂 PDF Upload & Analysis
* 🔑 MD5 / SHA1 / SHA256 Hash Generation
* 📑 Metadata Extraction
* 🏗 PDF Structure Analysis
* 🔍 Suspicious Keyword Detection
* ⚠ JavaScript Detection
* 🌐 IOC (Indicators of Compromise) Extraction
* 📊 Risk Scoring Engine
* 📄 HTML & PDF Report Generation
* 📱 Responsive Dashboard
* 📜 Analysis History
* 🌙 Modern Dark UI
